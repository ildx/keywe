# Keywe - A keyboard assistant

Keywe is a macOS native keyboard tool built with Swift.

## Features

1. **Hyper Key functionality** - Converts Caps Lock to a Hyper Key for advanced keyboard shortcuts
2. **Window tiling** - Efficient window management with keyboard shortcuts
3. **Application launching/switching** - Quick access to applications using keyboard shortcuts

## Setup

To set up the development environment:

1. Clone the repository
2. Install skills (optional):
   - Install Node.js (required for skills CLI)
   - Run `npx skills install`
3. [Build and run](docs/guides/build_and_run_instructions.md) the application
